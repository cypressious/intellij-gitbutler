<!-- Plugin description -->
Inofficial plugin for [GitButler](https://gitbutler.com/) support in IntelliJ IDEs. Requires the GitButler CLI to be in `PATH`.

### Features

- Set up and tear down Workspace
- Create New Virtual Branch
- Commit to Virtual Branch
- Apply Branch to Workspace
- Unapply Virtual Branch
- Delete Branch
- Push Virtual Branch
- Create PR
- Pull & Update Workspace

<!-- Plugin description end -->

## Installation

- Using the IDE built-in plugin system:

  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "intellij-gitbutler"</kbd> >
  <kbd>Install</kbd>

- Using JetBrains Marketplace:

  Go to [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/31299) and install it by clicking the <kbd>Install to ...</kbd> button in case your IDE is running.

  You can also download the [latest release](https://plugins.jetbrains.com/plugin/31299/versions) from JetBrains Marketplace and install it manually using
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>

- Manually:

  Download the [latest release](https://github.com/cypressious/intellij-gitbutler/releases/latest) and install it manually using
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>
